## 🌐 Deploy with GitHub Pages

To deploy this portfolio with GitHub Pages:

2. Scroll to "GitHub Pages" section
3. Select "main" branch as the source
4. Your portfolio will be live at: `https://katiefj123-bot.github.io/portfolio/`
